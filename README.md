# Kathryne #

Discord bot that emulates the Wish system in Genshin Impact.

------------

Clone and enter directory:

```terminal
git clone https://github.com/akomi-dev/Kathryne
cd Kathryn
```

Install requirements:

```terminal
npm install
```

Add your guild (server) id to publicConfig.json:

``` 
{
    ...
    "guildId": ""
}
```

Run script:

```terminal
node index.js
```
